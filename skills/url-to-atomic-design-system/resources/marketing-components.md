# Marketing Components

Marketing components are reusable conversion-oriented sections.

## Hero Components

### Hero / Split Visual

Use when the reference has text on one side and product, image, video, or visual composition on the other.

Anatomy:

- Eyebrow
- Headline
- Supporting paragraph
- CTA group
- Proof element
- Media area

### Hero / Centered

Use when the reference uses a central editorial or SaaS-style introduction.

Anatomy:

- Eyebrow
- Headline
- Paragraph
- CTA group
- Optional logo cloud

## Authority Components

### Authority / Logo Cloud

Used to show trust through clients, partners, certifications, or media mentions.

### Authority / Stats Row

Used to show numbers, experience, scale, or credibility.

## Benefit Components

### Benefits / Feature Grid

Anatomy:

- Section heading
- Supporting copy
- Feature cards
- Optional CTA

### Benefits / Alternating Rows

Used when each benefit deserves more explanation.

## Product Components

### Product / Showcase

Anatomy:

- Product image
- Product title
- Product description
- Feature list
- CTA

### Product / Card Grid

Used for multiple products or services.

## Conversion Components

### CTA / Conversion Strip

Anatomy:

- Headline
- Short copy
- Primary CTA
- Secondary CTA

### CTA / Lead Capture

Anatomy:

- Headline
- Form
- Privacy note
- Trust element

## Objection Handling Components

### FAQ / Accordion

Used to reduce friction before conversion.

### Comparison / Table

Used to differentiate against competitors or alternatives.

## Storytelling Components

### Process / Timeline

Used to explain steps.

### Case / Proof Block

Used to explain a client result or project success.
