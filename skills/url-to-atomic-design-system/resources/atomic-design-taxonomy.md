# Atomic Design Taxonomy

## Foundations

Foundations are not visible components by themselves. They define the design language.

Required foundations:

- Colors
- Typography
- Spacing
- Radius
- Shadows
- Borders
- Motion
- Grid
- Breakpoints
- Z-index
- Iconography
- Image style

## Atoms

Atoms are the smallest usable UI elements.

Common atoms:

- Button
- Link
- Input
- Textarea
- Select
- Checkbox
- Radio
- Label
- Badge
- Tag
- Icon
- Divider
- Logo
- Image
- Heading
- Paragraph
- Eyebrow
- Caption
- Avatar

## Molecules

Molecules combine atoms into functional UI units.

Common molecules:

- Form field
- Search input
- Navigation item
- CTA group
- Feature item
- Benefit item
- Product card
- Service card
- Testimonial card
- Stat item
- Accordion item
- Breadcrumb
- Newsletter signup
- Social proof row

## Organisms

Organisms are complete interface sections.

Common organisms:

- Header
- Footer
- Hero
- Feature grid
- Product showcase
- Testimonial section
- FAQ section
- Pricing table
- Contact section
- Blog grid
- Case study block
- Lead capture form

## Templates

Templates define page-level layouts.

Common templates:

- Landing page
- Homepage
- Product page
- Service page
- Category page
- Article page
- Case study page
- Contact page

## Marketing Components

Marketing components are reusable conversion modules.

Examples:

- Hero with proof
- Authority block
- Benefits stack
- Differentiator grid
- Comparison block
- Objection handling FAQ
- Conversion CTA strip
- Lead magnet module
- Offer block
- Process timeline
- Trust block
- Case proof module
